Video Game Sales - Can we Predict the Future?¶

Orel Mizrahi

Project Description¶

My project is going to be an analysis of video game sales data, where I will be attempting to find predictors of total global sales. The data set I'm using is called "Video Game Sales and Ratings" and can be found here: https://www.kaggle.com/datasets/thedevastator/video-game-sales-and-ratings.

Guiding Questions¶

There is a lot of useful data in this dataset, and with my project, I wanted to answer two questions

  1. Do Critic and User scores of a game predict sales?
  2. Which of the two predicts sales more closely?

But after my analysis of those questions, I also decided to explore a third possibility.

  1. Can ratings predict video game sales?.

Why does this matter?¶

I believe these questions have critical real-world applications and can affect game creation and marketing. For example, If it is found that games rated E for Everyone significantly outperform games rated M for Mature, it would stand to reason that game publishers would create more games rated E to give their game the best shot at doing well financially. Therefore, knowing the relationship between scores, ratings, and sales can change the decision-making for developers and alter the types of games that get created and released.

Analyzing Scores vs Sales¶

Now that all the data is cleaned up, I can start with the analysis. The first thing I'm going to do is make a graph plotting the scores given vs their global sales. I'm splitting this up into Critic vs User and putting them side by side to help visualize a difference and also answer my second question: Which of the two more closely predicts sales?

Two choices I made that I wanted to justify were the lines "axes[0].set_xlim(0, 100)" and "axes[0].set_yscale("log")". The easier to explain was setting the x-axis from 0 - 100 to standardize the view for both graphs, as they appeared different by default. The other change was using the log scale. For both of these graphs, there were outliers that heavily skewed the graphs, making it difficult to read and gain insight from. I had the choice to limit the y-axis and exclude some data or use a log view. I chose to use the log view as I felt excluding data would taint my analysis, although as I found out later on, my choice had its own consequences.

Linear Regression¶

I wanted to run a regression to get solid data on the predictions I made previously. I decided linear regression best fits my task to try to use the three variables I've been looking at to predict the sales outcome.

In [72]:
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

df_model = df_analysis.copy()

df_model = pd.get_dummies(df_model, columns=["Rating"], drop_first=True)
x = df_model.drop("Global_Sales", axis=1)
y = np.log1p(df_model["Global_Sales"])
x_train, x_test, y_train, y_test = train_test_split(
    x, y,
    test_size=0.2,
    random_state=42
)

model = LinearRegression()
model.fit(x_train, y_train)
y_pred = model.predict(x_test)
print("R^2 Score:", r2_score(y_test, y_pred))
print("RMSE:", np.sqrt(mean_squared_error(y_test, y_pred)))

R^2 Score: 0.19153106729401137
RMSE: 0.4433323978390105

(A log transformation of sales was applied to address extreme skews in the data)

Insight and Future Goals¶

Going back to my original questions, I now have my answers. Overall, it seems that critics are better at predicting sales compared to users, but that prediction has a low correlation of 0.239. In the data, low scores are good predictors of low sales, but high scores are not necessarily good predictors of high sales. My final question was whether or not ratings could predict video game sales. After my analysis, it seems that ratings do not predict sales.

To further evaluate this relationship, I created a linear regression model using critic score, user score, and ESRB rating to predict global sales. Because the sales data was heavily skewed by extreme outliers, I applied a logarithmic transformation to the sales values before training the model. The regression model achieved an (R^2) score of approximately 0.19, meaning the variables in the model explained about 19% of the variation in global sales. This indicates that review scores and ratings do have some predictive value, but they are not strong predictors on their own.

The choices I made regarding presentation and the presence of extremes in the data exaggerated the trends in the data and led me to make false assumptions. In the future, my goal is to make better decisions about data visualizations to avoid making the same mistake again

Going forward, I think a more comprehensive dataset is needed. This is because after cleaning the data, sometimes I only had 1 entry for a variable to work with, and the majority of my data had NaNs, which removed data such as retro games that more often had missing data, rendering my analysis incomplete.
